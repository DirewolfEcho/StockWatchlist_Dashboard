# 最新动态信息格式优化

## 更新时间: 2026-01-31 18:30

### 优化需求

将最新动态信息从"标题+概述"格式改为"核心动态列表"格式，直接展示3条互不重复的市场核心信息。

### 新格式要求

1. **无标题**: 不显示新闻标题，直接展示核心内容
2. **编号列表**: 使用1、2、3编号展示
3. **字数限制**: 每条最多50字
4. **内容类型**: 
   - 重要市场动作（新产品、战略合作、业务调整）
   - 市场反馈（股价变动、市场情绪）
   - 资金流向（南向/北向资金动向）
   - 股东增减持
   - 投资机构研报、评级调整
   - 重大负面新闻（监管处罚、业绩预警）

### 示例格式

参考bilibili的动态信息：

```
1. bilibili公司于1月27日首次官宣了IP"2026bilibili大年初一联欢会"，旨在打造全年龄段合家欢晚会，对市场产生积极作用

2. 数据显示，南向资金近期对bilibili持股呈现波动，但在1月26日当日增持了67.71万股

3. 有市场机构预测，bilibili 2025年第四季度营业收入可能在78.74至89.58亿元之间，同比增长1.8%至15.8%
```

### 技术实现

#### 后端变更

**文件**: `backend/app/services/news_data.py`

**核心函数**: `_translate_and_summarize_news()`

**实现逻辑**:
```python
# 1. 从英文新闻中提取核心信息
# 2. 使用LLM生成中文市场动态总结
# 3. 重点关注：
#    - 市场动作
#    - 资金流向
#    - 机构研报
#    - 股东变动
#    - 重大新闻
# 4. 控制字数在50字以内
# 5. 返回纯文本（无标题）
```

**LLM提示词**:
```
请从以下新闻中提取核心信息，生成一条简洁的市场动态总结（最多50字）。

重点关注以下类型的信息：
- 重要市场动作（如新产品发布、战略合作、业务调整）
- 市场反馈（如股价变动、市场情绪）
- 资金流向（如南向资金、北向资金动向）
- 股东增减持
- 大行或投资机构的最新研报、评级调整
- 重大负面新闻（如监管处罚、业绩预警）

请直接返回一句话总结，不要包含标题，不要超过50字。
```

**数据模型**:
```python
# NewsItem 保持不变
class NewsItem(BaseModel):
    title: str  # 现在为空字符串
    content: str  # 核心动态总结（最多50字）
```
